Task 0.1 – Khởi tạo project & cấu hình cơ bản

Mục tiêu: Có skeleton project chạy được, sẵn Tailwind, TS.

Việc cần làm:

Init project (Next.js + TypeScript hoặc Vite + React + TS).

Cấu hình Tailwind (base styles, container, breakpoints).

Thêm thư viện UI (nếu muốn): Radix UI + headless components hoặc shadcn/ui.

Thêm router (nếu không dùng Next).

BE-friendly:

Tạo file src/config/api.ts với API_BASE_URL dùng env (import.meta.env / process.env).

Chuẩn bị ApiClient (axios/fetch wrapper) nhưng chưa gọi BE thật.
